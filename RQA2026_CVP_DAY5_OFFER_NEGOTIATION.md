```

   - CEO职位: 北京地区¥50K-100K/月，平均¥70K

   - CEO张总: 当前年薪约¥80K，期望¥90K+高股权
   - CTO陈工: 当前年薪约¥70K，期望¥75K+技术股权

   - 总人力预算: ¥430K/月 (9人团队)

### 谈判核心要点

#### 股权激励谈判
```
第一阶段: 股权比例确定
- CEO股权: 3% (公司估值¥1亿时价值¥300K)
- CTO股权: 3% (对等重视技术领导力)
- 股权池: 总计6%，预留14%用于后续招聘

第二阶段: Vesting条件
- 服务期: 4年 (行业标准)
- 解锁节奏: 每年25% (1年解锁1%，共4年)
- 里程碑解锁: 达到关键目标额外解锁0.5%
- 加速条款: 公司融资或上市时加速解锁

第三阶段: 退出机制
- 正常离职: 已解锁股权保留，未解锁作废
- 非正常离职: 按比例收回股权
- IPO退出: 按市场价退出
- 收购退出: 按收购价退出
```

#### 奖金设计谈判
```
绩效奖金:
- 月度奖金: 基本工资20% (¥14K CEO, ¥13K CTO)
- 计算周期: 月度考核，次月发放
- 考核指标: 个人KPI + 团队贡献 + 公司业绩

年终奖金:
- 奖金基数: 2个月基本工资
- 发放条件: 年度KPI完成度80%以上
- 特殊奖励: 突破性成果额外奖励

技术奖金 (CTO专项):
- 开源贡献奖: ¥50K (重大开源项目)
- 技术突破奖: ¥30K (核心技术突破)
- 专利申请奖: ¥20K (发明专利授权)
```

#### 福利待遇谈判
```
基础福利:
- 五险一金: 标准缴纳，无争议
- 商业保险: ¥50K保额，标准配置
