## 🛠️ 使用指南

### 基本用法
```bash
# 运行覆盖率测试
pytest --cov=src/your_package --cov-report=html --cov-report=term

# 设置覆盖率阈值
pytest --cov=src/your_package --cov-fail-under=80

# 生成多种报告格式
pytest --cov=src/your_package \
       --cov-report=html:htmlcov \
       --cov-report=json:coverage.json \
       --cov-report=xml:coverage.xml
```

### 配置建议
```ini
# pytest.ini
[tool:pytest]
addopts =
    --cov=src/your_package
    --cov-report=html
    --cov-report=term-missing
    --cov-fail-under=80
```

### CI/CD集成
```yaml
# GitHub Actions示例
- name: Run tests with coverage
  run: |
    pytest --cov=src --cov-report=xml --cov-fail-under=80
- name: Upload coverage
  uses: codecov/codecov-action@v3
  with:
    file: ./coverage.xml
```

## 🔍 技术细节

### 覆盖率计算公式
```
覆盖率 = (已覆盖行数 / 总语句数) × 100%
```

### 数据来源
- **总语句数**: 代码中可执行语句的数量
- **已覆盖行数**: 测试执行时实际执行的语句数
- **排除行数**: 配置中明确排除的代码行
