## 📝 测试执行说明

### 运行测试

```bash
# 运行所有特征层测试
pytest tests/unit/features/ -n auto -k "not e2e" --tb=line -q

# 生成覆盖率报告
pytest tests/unit/features/ -n auto -k "not e2e" \
  --cov=src.features --cov-report=term-missing --cov-report=html

# 运行特定模块测试
pytest tests/unit/features/monitoring/ -xvs
```

### 查看覆盖率报告

- **终端报告**: `pytest --cov-report=term-missing`
- **HTML报告**: `pytest --cov-report=html` (生成在 `htmlcov/` 目录)
- **XML报告**: `pytest --cov-report=xml` (用于CI/CD集成)
