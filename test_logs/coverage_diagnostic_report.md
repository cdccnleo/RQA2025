#### ✅ 修复3: 删除死锁测试
- **文件**: `tests/unit/infrastructure/health/test_60_ultimate_push.py`

### 3. 剩余58个失败测试分类

#### 类别A: API不匹配（约20个）
**典型错误**:
```python

```
**文件**: `test_backtest_monitor_plugin_comprehensive.py`
**原因**: 测试期望的API与实际实现不符

#### 类别B: 异步方法未await（约15个）
**典型错误**:
```python

```
**文件**: 多个测试文件
**原因**: 测试调用了async方法但未使用await

#### 类别C: 数据验证错误（约10个）
**典型错误**:
```python

```
**原因**: 测试断言与实际数据不符
