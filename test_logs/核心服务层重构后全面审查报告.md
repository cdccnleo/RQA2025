**代码示例**:
```python
# 优雅的模板方法模式
class BaseComponent:
    def execute(self, *args, **kwargs):
        # 统一流程控制
        if not self.is_initialized():
            raise RuntimeError("未初始化")
        
        try:
            self._status = ComponentStatus.RUNNING
            result = self._do_execute(*args, **kwargs)  # 调用子类实现
            self._status = ComponentStatus.INITIALIZED
            return result
        except Exception as e:
            self._handle_error(e)  # 统一错误处理
```

**代码示例**:

```python
# 优雅的适配流程
def adapt(self, data: InputType) -> OutputType:
    # 1. 检查缓存
    if self.enable_cache and cached:
        return cached_result
    
    # 2. 验证输入
    if not self.validate_input(data):
        raise ValueError("无效输入")
    
    # 3. 预处理
    preprocessed = self._preprocess(data)
    
    # 4. 适配
    result = self._do_adapt(preprocessed)
    
    # 5. 后处理
    postprocessed = self._postprocess(result)
    
    # 6. 缓存结果
    return postprocessed
```
