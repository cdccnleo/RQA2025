### 核心策略（已验证有效）

**1. 标准化测试模式**
```python
# 统一的测试结构
class TestComponent:
    def test_init(self):
        """测试初始化"""
        try:
            from module import Component
            obj = Component()
            assert isinstance(obj, Component)
        except ImportError:
            pytest.skip("Component not available")
```

**特点：**
- ✅ 容错设计（pytest.skip）
- ✅ 灵活适配（hasattr检查）
- ✅ 全面覆盖（初始化、功能、边界、集成）

**3. 精准定位目标**

**4. 快速迭代验证**

- 及时调整策略
