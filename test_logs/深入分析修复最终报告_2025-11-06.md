## 💡 关键经验

### ✅ 有效的方法

1. **诊断脚本**
   ```python
   # analyze_coverage_gap.py
   - 逐模块统计0%文件
   - 分析测试执行率
   - 识别文件名不匹配
   ```

2. **修复文件名**
   ```bash
   # 使用grep找实际类名
   grep "class.*SearchEngine" src/.../*.py
   
   # 然后修正导入
   from xxx_refactored import XXX  # ❌
   from xxx import XXX              # ✅
   ```

3. **移除不当skip**
   ```python
   # 不要这样：
   try:
       from xxx import XXX
   except:
       pytest.skip()  # ❌
   
   # 应该这样：
   from xxx import XXX  # ✅ 让错误暴露
   ```

4. **清理覆盖率文件**
   ```python
   # 每次测试前清理
   for cov_file in Path.cwd().glob('.coverage*'):
       cov_file.unlink()
   ```
