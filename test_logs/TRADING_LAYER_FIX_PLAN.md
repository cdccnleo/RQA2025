**目标**: 消除所有errors，确保所有测试可以运行

**常见错误类型**:
1. ImportError - 导入错误
2. AttributeError - 属性缺失
3. TypeError - 类型错误
4. NameError - 名称错误
5. ModuleNotFoundError - 模块未找到

**修复方法**:
1. 按错误类型分组
2. 批量修复同类错误
3. 验证修复效果
4. 继续下一个错误类型
