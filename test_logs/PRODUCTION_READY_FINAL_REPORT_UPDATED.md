## 📝 测试执行说明

### 运行测试

```powershell
# 运行特征层测试
pytest tests/unit/features/ -n auto -k "not e2e" --tb=line -q

# 运行模型层测试
pytest tests/unit/ml/ -n auto -k "not e2e" --tb=line -q

# 运行所有测试
pytest tests/unit/ -n auto -k "not e2e" --tb=line -q

# 生成覆盖率报告
pytest tests/unit/features/ -n auto -k "not e2e" \
  --cov=src.features --cov-report=term-missing

pytest tests/unit/ml/ -n auto -k "not e2e" \
  --cov=src.ml --cov-report=term-missing
```

### 查看覆盖率报告

```powershell
# 终端报告
pytest --cov-report=term-missing

# HTML报告（生成在 htmlcov/ 目录）
pytest --cov-report=html
```

### 保存测试日志

```powershell
# 使用简化脚本（推荐）
.\test_logs\run_tests_simple.ps1

# 直接使用命令
$timestamp = Get-Date -Format 'yyyyMMdd_HHmmss'
$logFile = "test_logs\test_statistics_$timestamp.log"
pytest tests/unit/ -n auto -k "not e2e" --tb=line -q --cov-report=term-missing 2>&1 | Out-File -FilePath $logFile -Encoding UTF8
```
