### 核心子系统

1. **核心模块** (`core/`) - 特征工程核心
   - FeatureEngine: 特征引擎
   - FeatureEngineer: 特征工程师
   - FeatureManager: 特征管理器
   - FeatureConfig: 特征配置
   - FeatureStore: 特征存储

2. **处理器模块** (`processors/`) - 特征处理
   - FeatureProcessor: 特征处理器
   - FeatureSelector: 特征选择器
   - FeatureStandardizer: 特征标准化器
   - TechnicalIndicatorProcessor: 技术指标处理器
   - 编码器、标准化器、转换器等

3. **技术指标模块** (`indicators/`) - 技术指标计算
   - ATR计算器
   - Bollinger计算器
   - CCI计算器
   - Fibonacci计算器
   - Ichimoku计算器
   - KDJ计算器
   - Momentum计算器
   - Volatility计算器
   - Williams计算器

4. **工程模块** (`engineering/`) - 特征工程
   - Builder组件: 特征构建器
   - Creator组件: 特征创建器
   - Extractor组件: 特征提取器
   - Generator组件: 特征生成器
   - Engineer组件: 特征工程师

5. **加速模块** (`acceleration/`) - 硬件加速
   - GPU加速: GPU加速组件
   - FPGA加速: FPGA加速组件
   - 分布式加速: 分布式加速组件
   - 性能优化: 性能优化组件

6. **分布式模块** (`distributed/`) - 分布式处理
   - DistributedProcessor: 分布式处理器
   - TaskScheduler: 任务调度器
   - WorkerManager: 工作管理器

7. **监控模块** (`monitoring/`) - 特征监控
   - FeaturesMonitor: 特征监控器
   - PerformanceAnalyzer: 性能分析器
   - MetricsCollector: 指标收集器
   - AlertManager: 告警管理器

8. **存储模块** (`store/`) - 特征存储
   - CacheStore: 缓存存储
   - DatabaseStore: 数据库存储
   - PersistenceStore: 持久化存储
   - RepositoryStore: 仓库存储

9. **智能模块** (`intelligent/`) - 智能特征
   - AutoFeatureSelector: 自动特征选择器
   - IntelligentEnhancementManager: 智能增强管理器
   - MLModelIntegration: ML模型集成
   - SmartAlertSystem: 智能告警系统

10. **订单簿模块** (`orderbook/`) - 订单簿分析
    - OrderBookAnalyzer: 订单簿分析器
    - Level2Analyzer: Level2分析器
    - OrderBookMetrics: 订单簿指标

11. **情感分析模块** (`sentiment/`) - 情感分析
    - SentimentAnalyzer: 情感分析器
    - 情感模型: 多种情感分析模型

12. **插件模块** (`plugins/`) - 插件系统
    - PluginManager: 插件管理器
    - PluginLoader: 插件加载器
    - PluginRegistry: 插件注册表

13. **性能模块** (`performance/`) - 性能优化
    - PerformanceOptimizer: 性能优化器
    - HighFreqOptimizer: 高频优化器
    - ScalabilityManager: 可扩展性管理器
