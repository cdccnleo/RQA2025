```ini

# 增强的标记描述
unit: 快速单元测试（<5秒，每次提交运行）
integration: 集成测试（<30秒，PR合并前运行）
performance: 性能基准测试（手动/定期执行，默认跳过）
slow: 慢速测试（>5秒，需要优化或重新分类）
flaky: 不稳定测试（需要修复，默认跳过）
smoke: 冒烟测试（快速验证核心功能）
```
