## 💡 核心策略（已验证）

### 最高效的测试类型

1. **模块级函数测试** (10-16行/测试) ⭐⭐⭐⭐⭐
   ```python
   def test_module_health_check():
       from some_module import health_check
       result = health_check()
       assert result["healthy"]
   ```

2. **集成场景测试** (5-10行/测试) ⭐⭐⭐⭐
   ```python
   def test_integration():
       comp1, comp2 = Comp1(), Comp2()
       comp1.register(comp2)
       result = comp1.execute()
       assert len(result) > 0
   ```

3. **业务逻辑测试** (3-6行/测试) ⭐⭐⭐
   ```python
   def test_workflow():
       obj = Class()
       for i in range(10): obj.add(i)
       assert len(obj.get()) == 10
   ```

### 避免低效测试

❌ **简单初始化测试** (~1行/测试)
```python
def test_init():
    obj = Class()
    assert obj is not None
```
