```
   - 展现对候选人的重视
   - 准备庆祝录用成功的方案

```

```
   - 确认录用意愿和兴奋度

   - 了解对福利的需求
   - 确认股权激励的看法
```

```
1. 月薪谈判

   - 讨论绩效奖金计算

   - 确定个性化福利方案

4. 处理异议和顾虑
   - 回答薪资相关疑问
   - 解释股权激励机制
   - 打消福利待遇顾虑
```

```
   - 说明后续手续流程

2. 签署初步协议
```

```
- 应对: 设定时间节点，保持谈判节奏
```
