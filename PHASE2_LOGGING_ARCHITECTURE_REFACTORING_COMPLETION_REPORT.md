#### 新目录结构创建
```
src/infrastructure/logging/
├── core/           # 核心接口和基类
│   ├── __init__.py     # 统一导入
│   ├── interfaces.py   # 接口定义
│   └── exceptions.py   # 异常定义
├── handlers/       # 日志处理器
│   ├── __init__.py     # 统一导入
│   ├── base.py         # 基础处理器
│   ├── console.py      # 控制台处理器
│   ├── file.py         # 文件处理器
│   └── remote.py       # 远程处理器
├── formatters/     # 日志格式化器
│   ├── __init__.py     # 统一导入
│   ├── base.py         # 基础格式化器
│   ├── text.py         # 文本格式化器
│   ├── json.py         # JSON格式化器
│   └── structured.py   # 结构化格式化器
├── storage/        # 日志存储
│   ├── __init__.py     # 统一导入
│   └── base.py         # 基础存储
├── monitors/       # 监控组件
│   ├── __init__.py     # 统一导入
│   └── base.py         # 基础监控器
└── services/       # 服务层
    ├── __init__.py         # 统一导入
    ├── base_service.py     # 基础服务
    └── logger_service.py   # 日志服务
```

#### 目录职责明确
- **core**: 核心接口、异常、全局配置
- **handlers**: 日志输出处理器
- **formatters**: 日志格式化逻辑
- **storage**: 日志持久化存储
- **monitors**: 系统监控和健康检查
- **services**: 业务服务封装
