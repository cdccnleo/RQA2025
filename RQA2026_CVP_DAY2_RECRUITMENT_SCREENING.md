## 📊 招聘初筛标准

### CEO候选人筛选标准
#### ✅ 必须满足 (淘汰不符合项)
- 3年以上创业或高管经验
- 金融科技/量化交易行业背景
- 本科以上学历，知名高校优先
- 年龄35-55岁，精力充沛

#### 🔄 加分项 (优先考虑)
- 成功融资经验 (天使轮/A轮)
- AI技术相关创业经验
- 海外留学或工作经验
- 优秀的演讲和沟通能力

#### ❌ 淘汰标准
- 无创业经验，仅有打工经历
- 非金融科技行业背景
- 缺乏领导力和管理经验
- 年龄过大或过小

### CTO候选人筛选标准
#### ✅ 必须满足 (淘汰不符合项)
- 5年以上技术团队管理经验
- 架构设计和系统开发经验
- 金融科技系统开发经验
- 硕士以上学历，计算机相关专业

#### 🔄 加分项 (优先考虑)
- AI/ML算法开发经验
- 云原生架构经验 (AWS/Azure)
- 开源项目贡献或技术影响力
- 英语流利，可与国际团队沟通

#### ❌ 淘汰标准
- 缺乏架构设计经验
- 无团队管理经验
- 非技术背景或经验不足
- 缺乏金融行业理解

## ☁️ AWS环境验证清单

### 网络连接验证
```bash
# VPN连接测试
ping 10.0.0.1  # VPC内部IP
traceroute rqa2026-dev-vpc.com

# DNS解析测试
nslookup eks.rqa2026.com
nslookup rds.rqa2026.com

# 安全组规则验证
aws ec2 describe-security-groups --group-ids sg-12345678
```

### EKS集群验证
```bash
# 集群状态检查
aws eks describe-cluster --name rqa2026-dev-cluster

# 节点组验证
kubectl get nodes
kubectl get pods --all-namespaces

# GPU节点检查
kubectl get nodes -l node-type=gpu
```

### RDS数据库验证
```bash
# 连接测试
psql -h rqa2026-dev-db.cluster-xxxx.rds.amazonaws.com \
     -U rqa2026_admin \
     -d rqa2026_dev

# 数据库状态检查
SELECT version();
SELECT current_database();
```

### S3存储验证
```bash
# 存储桶访问测试
aws s3 ls s3://rqa2026-dev-data/
aws s3 ls s3://rqa2026-dev-models/
aws s3 ls s3://rqa2026-dev-logs/

# 权限验证
aws s3api get-bucket-versioning --bucket rqa2026-dev-data
```
