## 🚀 使用指南

### 本地开发环境设置

1. **安装pre-commit钩子**：
```bash
pip install pre-commit
pre-commit install
```

2. **运行代码质量检查**：
```bash
make quality
```

3. **格式化代码**：
```bash
make format
```

### CI/CD 流水线

项目配置了完整的GitHub Actions CI/CD流水线，包括：
- 代码质量检查（flake8）
- 单元测试和集成测试
- 覆盖率报告
- 自动部署

### 代码质量标准

项目采用以下代码质量标准：
- **Python版本**: 3.9+
- **最大行长**: 100字符
- **代码复杂度**: ≤12
- **测试覆盖率**: ≥15%
- **代码风格**: Black + flake8
