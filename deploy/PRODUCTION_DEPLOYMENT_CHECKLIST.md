# 生产环境部署检查清单

## 概述

本文档提供了生产环境部署前的完整检查清单，确保所有必要的准备工作都已完成，降低部署风险。

## 部署前检查清单

### 🔧 基础设施检查

#### 服务器资源
- [ ] **CPU资源**
  - [ ] 确认CPU核心数满足要求 (推荐: 8核心以上)
  - [ ] 确认CPU负载在正常范围内 (< 70%)
  - [ ] 确认CPU预留资源充足

- [ ] **内存资源**
  - [ ] 确认可用内存满足要求 (推荐: 16GB以上)
  - [ ] 确认内存使用率在正常范围内 (< 80%)
  - [ ] 确认swap空间配置合理

- [ ] **磁盘资源**
  - [ ] 确认磁盘空间充足 (推荐: 100GB以上可用空间)
  - [ ] 确认磁盘I/O性能满足要求
  - [ ] 确认磁盘分区配置合理

#### 网络配置
- [ ] **网络连通性**
  - [ ] 确认服务器网络连通性正常
  - [ ] 确认防火墙规则配置正确
  - [ ] 确认负载均衡器配置正确
  - [ ] 确认DNS解析正常

- [ ] **端口配置**
  - [ ] 确认应用端口 (8080) 未被占用
  - [ ] 确认数据库端口 (5432) 可访问
  - [ ] 确认Redis端口 (6379) 可访问
  - [ ] 确认监控端口 (9090, 3000) 可访问

#### 安全配置
- [ ] **访问控制**
  - [ ] 确认SSH密钥配置正确
  - [ ] 确认防火墙规则限制访问

- [ ] **加密配置**

## 检查清单使用说明

### 使用方法
1. **部署前**: 逐项检查所有检查项，确保全部完成
2. **部署中**: 实时监控部署进度，及时处理问题
3. **部署后**: 全面验证部署结果，确保系统正常运行

### 检查结果记录
- 每个检查项完成后，在方框中打勾 [x]
- 对于未完成的检查项，记录具体原因和计划完成时间
- 对于有问题的检查项，记录问题描述和解决方案

### 检查清单更新
- 根据实际部署经验，定期更新检查清单
- 添加新的检查项，删除过时的检查项
- 优化检查项的描述和分类
