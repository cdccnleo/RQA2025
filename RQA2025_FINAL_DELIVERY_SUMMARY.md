## 🚀 快速部署指南

### 环境准备
```bash
# 1. 解压交付包
unzip rqa2025_complete_delivery_*.zip

# 2. 进入项目目录
cd rqa2025_source/

# 3. 创建Python环境
python -m venv venv && source venv/bin/activate

# 4. 安装依赖
pip install -r requirements.txt
```

### 基础设施启动
```bash
# 5. 启动数据库和缓存
docker-compose up -d

# 6. 运行数据库迁移
python scripts/migrate_database.py

# 7. 配置环境变量
cp .env.example .env  # 根据需要修改配置
```

### 应用部署
```bash
# 8. 启动应用服务
python -m src.main

# 9. 验证部署
curl http://localhost:8000/health

# 10. 运行测试验证
python tests/run_enhanced_test_suite.py
```
