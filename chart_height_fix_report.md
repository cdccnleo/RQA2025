# 数据源错误率和可用性统计Chart高度优化报告

## 🔍 问题诊断

### 用户反馈
**数据源错误率统计和数据源可用性统计仪表盘chart可以增加高度，下方有较多空白**

### 问题分析
- 数据源错误率统计和数据源可用性统计两个chart的高度设置为200px
- 下方存在较多空白区域，没有充分利用页面空间
- chart内容相对简单，高度200px显得过于紧凑

## ✅ 修复方案

### 修复位置
**文件**: `web-static/data-sources-config.html`

### 修改内容
将两个chart的高度从200px增加到300px：

#### 1. 错误率统计Chart
**位置**: 第675行
```html
<!-- 修改前 -->
<canvas id="errorRateChart" width="400" height="200"></canvas>

<!-- 修改后 -->
<canvas id="errorRateChart" width="400" height="300"></canvas>
```

#### 2. 可用性统计Chart
**位置**: 第688行
```html
<!-- 修改前 -->
<canvas id="availabilityChart" width="400" height="200"></canvas>

<!-- 修改后 -->
<canvas id="availabilityChart" width="400" height="300"></canvas>
```
