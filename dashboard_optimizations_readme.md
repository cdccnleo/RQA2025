# 数据源配置管理页面短期优化功能

## 🎯 优化概述

按照 `dashboard_data_source_fix_report.md` 中的建议，完成了三个短期优化功能：

1. **数据源过滤功能** - 允许用户选择显示特定类型的数据源
2. **图表数据缓存** - 减少API请求频率，提升性能
3. **数据源状态指示器** - 实时显示数据源连接状态

## 📋 功能详情

### 1. 数据源类型过滤功能

**位置**: 仪表盘顶部 "数据源类型过滤" 区域

**功能特性**:
- 显示所有可用的数据源类型（股票数据、债券数据、期货数据、外汇数据、宏观经济、财经新闻等）
- 支持多选：可以选择多个类型进行组合过滤
- 快捷按钮：全选、清空、应用过滤
- 实时反馈：显示已选择的类型数量

**使用方法**:
1. 在过滤区域勾选想要显示的数据源类型
2. 点击"应用过滤"按钮
3. 图表将只显示选中类型的数据源数据

### 2. 图表数据缓存功能

**位置**: 仪表盘顶部 "图表数据状态" 区域

**功能特性**:
- 自动缓存API数据，缓存时间5分钟
- 显示缓存状态（有效/过期/空缓存）
- 显示最后更新时间和过期倒计时
- 提供手动刷新和清空缓存功能
- 减少不必要的API请求，提升页面响应速度

**缓存状态指示**:
- 🟢 **有效**: 显示"X分钟前更新，Y分钟后过期"
- 🟡 **过期**: 显示"已过期X分钟"
- 🔘 **空缓存**: 显示"无缓存数据"

**操作按钮**:
- **刷新**: 强制从服务器获取最新数据
- **清缓存**: 清空本地缓存，下次请求将从服务器获取

### 3. 数据源状态指示器

**位置**: 仪表盘顶部 "数据源状态指示器" 区域

**功能特性**:
- 实时显示所有数据源的连接状态
- 颜色编码状态指示：
  - 🟢 **绿色**: 连接正常
  - 🔴 **红色**: 连接异常
  - 🟡 **黄色**: 连接超时
  - ⚪ **灰色**: 未测试/已禁用
- 状态统计摘要：显示各类状态的数据源数量
- 自动刷新：每30秒自动更新状态
- 手动刷新：点击"刷新状态"按钮立即更新

**状态指示器布局**:
- 顶部显示状态统计（正常/异常/超时/未测试/已禁用数量）
- 网格布局显示每个数据源的状态指示器
- 悬停显示详细信息（数据源名称、类型、当前状态）

## 🔧 技术实现

### 数据源过滤
```javascript
// 获取过滤后的数据源
function getFilteredDataSources(allSources, selectedTypes) {
    return allSources.filter(source => {
        const sourceType = source.type || 'unknown';
        return selectedTypes.has(sourceType);
    });
}
```

### 图表缓存
```javascript
// 缓存管理
let chartDataCache = {
    expiryTime: 5 * 60 * 1000, // 5分钟
    timestamp: null,
    lastUpdate: null
};

// 检查缓存有效性
function isCacheValid() {
    const age = Date.now() - chartDataCache.timestamp;
    return age < chartDataCache.expiryTime;
}
```

### 状态指示器
```javascript
// 状态映射
function getStatusIndicatorClass(status) {
    if (status.includes('连接正常')) return 'bg-green-500';
    if (status.includes('连接异常')) return 'bg-red-500';
    if (status.includes('连接超时')) return 'bg-yellow-500';
    return 'bg-gray-400'; // 未测试
}
```
