# RQA2025用户操作手册

## 1. 概述

RQA2025是专为量化交易和风险管理设计的智能分析平台。本手册将指导用户如何使用系统的各项功能。

### 1.1 主要功能
- 🔐 **多因素认证**：安全可靠的用户身份验证
- 🛡️ **数据保护**：敏感信息自动脱敏和加密
- 📊 **智能监控**：实时系统性能和风险监控
- 📈 **交易分析**：全面的交易数据分析工具

### 1.2 目标用户
- 量化交易员
- 风险管理人员
- 系统管理员
- 业务分析师

## 2. 系统登录

### 2.1 访问地址
```
生产环境: https://rqa2025.example.com
测试环境: https://test-rqa2025.example.com
```

### 2.2 登录步骤

1. **打开浏览器**，访问系统地址
2. **输入用户名和密码**
3. **验证动态验证码**（TOTP）
4. **点击登录按钮**

### 2.3 多因素认证说明

系统采用密码 + TOTP（时间-based一次性密码）的双因子认证方式：

- **密码要求**：
  - 长度至少8位
  - 包含大写字母、小写字母、数字和特殊字符
  - 定期更换密码

- **TOTP设置**：
  1. 首次登录后，系统会提示设置TOTP
  2. 使用手机TOTP应用（如Google Authenticator）扫描二维码
  3. 保存备份密钥

### 2.4 忘记密码处理

1. 点击登录页面的"忘记密码"链接
2. 输入注册邮箱地址
3. 检查邮箱中的重置链接
4. 按照提示设置新密码

## 3. 系统功能使用

### 3.1 仪表板

登录后首先进入系统仪表板，显示：
- **系统状态**：CPU使用率、内存使用率、磁盘使用率
- **活跃告警**：当前系统告警信息
- **关键指标**：交易量、成功率、响应时间等

#### 仪表板操作
- **刷新数据**：点击右上角刷新按钮
- **查看详情**：点击各项指标卡片查看详细数据
- **设置告警**：点击告警项进行配置

### 3.2 数据管理

#### 数据上传
1. 进入"数据管理" -> "数据上传"
2. 选择文件类型（CSV、Excel、JSON）
3. 选择数据分类（交易数据、用户信息、市场数据）
4. 点击"上传"按钮

#### 数据保护说明
系统自动对上传的数据进行保护：
- **手机号**：显示为 `138****5678`
- **邮箱**：显示为 `zh****@example.com`
- **银行账户**：转换为令牌存储
- **其他敏感信息**：自动识别并保护

#### 数据查询
1. 进入"数据管理" -> "数据查询"
2. 选择查询条件
3. 查看保护后的数据
4. 导出查询结果（受保护状态）

### 3.3 监控告警

#### 查看告警
1. 进入"监控告警" -> "告警列表"
2. 查看活跃告警和历史告警
3. 按级别、时间、类型筛选

#### 告警级别说明
- **🟢 INFO**：信息提示，无需处理
- **🟡 WARNING**：警告信息，需要关注
- **🔴 ERROR**：错误信息，需要处理
- **🚨 CRITICAL**：严重错误，立即处理

#### 告警处理流程
1. 收到告警通知（邮件/控制台）
2. 查看告警详情
3. 根据处理指南进行处理
4. 确认告警解决

### 3.4 个人设置

#### 修改密码
1. 点击右上角用户名 -> "个人设置"
2. 选择"修改密码"
3. 输入当前密码和新密码
4. 验证动态验证码

#### 设置通知偏好
1. 进入"个人设置" -> "通知设置"
2. 选择通知渠道（邮件、短信）
3. 设置通知时间和频率
4. 保存设置

## 4. 常见问题解答

### 4.1 登录问题

**Q: 忘记密码怎么办？**
A: 点击"忘记密码"链接，通过邮箱重置。

**Q: TOTP验证码无效？**
A: 检查手机时间是否正确，TOTP有效期为30秒。

**Q: 账户被锁定？**
A: 联系系统管理员解锁，或等待30分钟自动解锁。

### 4.2 数据问题

**Q: 上传的数据为什么显示不完整？**
A: 系统自动对敏感信息进行保护，原始数据安全存储。

**Q: 如何查看原始数据？**
A: 普通用户无法查看原始数据，需管理员权限。

**Q: 数据导出格式是什么？**
A: 支持CSV、Excel、JSON格式，敏感数据保持保护状态。

### 4.3 监控问题

**Q: 收到告警怎么处理？**
A: 查看告警详情，根据处理指南进行操作。

**Q: 如何设置自定义告警？**
A: 联系系统管理员或使用高级用户权限设置。

**Q: 告警通知没有收到？**
A: 检查邮箱设置和通知偏好配置。

### 4.4 性能问题

**Q: 系统响应慢怎么办？**
A: 检查网络连接，避开高峰时段，或联系技术支持。

**Q: 数据上传失败？**
A: 检查文件格式和大小限制（最大100MB）。

**Q: 页面显示异常？**
A: 刷新页面，清除浏览器缓存，或使用其他浏览器。

## 5. 安全使用指南

### 5.1 密码安全
- 定期更换密码（每3个月）
- 不要使用与其他系统相同的密码
- 不要将密码告诉他人

### 5.2 数据安全
- 上传数据前进行脱敏处理
- 不要上传包含敏感信息的文件
- 下载数据时注意保护

### 5.3 操作安全
- 及时登出系统
- 不要在公共设备上使用
- 发现异常立即报告

### 5.4 网络安全
- 使用安全的网络环境
- 避免在公共WiFi下进行敏感操作
- 定期更新浏览器和操作系统
