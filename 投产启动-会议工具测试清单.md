### 5. 故障排除指南

#### 5.1 常见问题处理

| 问题 | 可能原因 | 解决方案 | 备用方案 |
|------|---------|---------|---------|
| **投影仪无信号** | HDMI线松动 / 分辨率不匹配 | 检查线缆 / 调整分辨率 | 用笔记本直接展示 / 打印材料 |
| **麦克风无声音** | 设备未选 / 静音 | 检查音频输入 / 取消静音 | 用手机录音 / 备用麦克风 |
| **网络中断** | WiFi不稳 | 切换到网线 / 手机热点 | 暂停会议 / 切换到线下讨论 |
| **Zoom无法进入** | 密码错误 / 网络问题 | 检查密码 / 刷新页面 | 用浏览器参会 / 切换Teams |
| **PPT打不开** | 文件损坏 / 版本不兼容 | 用其他电脑打开 / 转换格式 | 用PDF版本 / 口头说明 |
