### LRU淘汰算法
```python
# 使用OrderedDict实现O(1) LRU淘汰
def _evict_lru(self):
    """LRU淘汰策略"""
    if len(self._cache) >= self.max_size:
        key, _ = self._cache.popitem(last=False)  # 淘汰最少使用的
        self._remove(key)
        self._evictions += 1
```
