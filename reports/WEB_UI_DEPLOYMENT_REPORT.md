## 🚀 部署配置

### Docker容器配置
```yaml
rqa2025-web:
  image: nginx:alpine
  ports:
    - "8080:8080"
  volumes:
    - ./web-static/index.html:/usr/share/nginx/html/index.html:ro
    - ./web-static/nginx.conf:/etc/nginx/conf.d/default.conf:ro
  networks:
    - rqa2025
  healthcheck:
    test: ["CMD", "curl", "-f", "http://localhost:8080/health"]
```

### Nginx配置特点
- **静态文件服务**: 提供HTML/CSS/JS文件
- **API代理转发**: `/api/*` 请求转发到后端服务
- **CORS支持**: 跨域请求支持
- **安全加固**: 安全headers配置
- **性能优化**: Gzip压缩和缓存配置
