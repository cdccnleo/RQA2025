## 🛠️ 具体修复方案

### 方案一: 异步测试支持
```python
# 在测试文件开头添加
import pytest_asyncio

# 或者在pytest.ini中配置
# pytest.ini
[tool:pytest]
asyncio_mode = auto
```

### 方案二: 修复测试类
```python
# 错误示例
class TestDataManager:
    def __init__(self):
        pass

# 正确示例
class TestDataManager:
    @pytest.fixture
    def data_manager(self):
        return DataManager()
```

### 方案三: 修复返回值
```python
# 错误示例
def test_access_control():
    result = check_permission()
    return result  # 警告

# 正确示例
def test_access_control():
    result = check_permission()
    assert result is True  # 推荐
```
