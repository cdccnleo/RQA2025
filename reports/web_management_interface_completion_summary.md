## 🏗️ 系统架构

### 核心技术栈
- **后端框架**: Flask (Python Web框架)
- **实时通信**: Flask-SocketIO (WebSocket支持)
- **前端框架**: Bootstrap 5 (响应式设计)
- **数据可视化**: Chart.js (图表库)
- **模板引擎**: Jinja2 (动态页面生成)

### 系统组件
```
WebManagementInterface
├── Flask应用 (Flask App)
├── SocketIO (实时通信)
├── API路由 (RESTful API)
├── 页面路由 (HTML页面)
├── 后台更新线程 (数据更新)
└── 模板系统 (HTML模板)
```

### 5. 实时数据更新
- **WebSocket通信**: 实时数据推送

## 🔧 部署配置

### 环境要求
- **Python**: 3.7+
- **依赖包**: Flask, Flask-SocketIO, psutil
- **浏览器**: 现代浏览器支持WebSocket

### 启动方式
```python
from src.infrastructure.performance import WebManagementInterface

# 创建实例
interface = WebManagementInterface()

# 启动Web界面
interface.start_web_interface(host='0.0.0.0', port=5000)
```
