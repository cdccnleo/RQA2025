### 配置类核心特性

#### ✨ 特性1: 统一验证框架
```python
@dataclass
class TestSuiteConfig(BaseConfig):
    def _validate_impl(self, result: ValidationResult):
        if not self.suite_id:
            result.add_error("套件ID不能为空")
        
        # 递归验证子配置
        for scenario in self.scenarios:
            result.merge(scenario.validate())

# 自动验证
config = TestSuiteConfig(...)  # __post_init__自动验证，错误即抛出异常
```

#### ✨ 特性2: 类型安全
```python
# 完整的类型注解
@dataclass
class EndpointConfig(BaseConfig):
    path: str
    method: str
    operation_id: str
    parameters: List[EndpointParameterConfig]  # 强类型
    responses: List[EndpointResponseConfig]    # 强类型
```

#### ✨ 特性3: 序列化支持
```python
config = EndpointConfig(...)

# 转为字典
config_dict = config.to_dict()

# 从字典创建
new_config = EndpointConfig.from_dict(config_dict)
```

#### ✨ 特性4: 丰富的辅助方法
```python
# FlowGenerationConfig
config.add_node(node)
config.add_edge(edge)
config.get_node(node_id)

# TestSuiteConfig
config.add_scenario(scenario)
config.get_all_test_cases()
config.count_test_cases()  # 返回统计信息

# SchemaGenerationConfig
config.add_schema(schema, category='data')
config.get_all_schemas()
config.count_schemas()  # 按类别统计
```
