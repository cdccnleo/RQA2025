## 🎯 推荐验证流程

### 日常快速验证

```powershell
# 1. 快速运行所有测试（安静模式）
pytest tests/unit/ -q

# 2. 如果有失败，详细查看
pytest tests/unit/ -v --tb=short
```

### 投产前完整验证

```powershell
# 1. 运行全部测试并生成覆盖率报告
pytest tests/unit/ --cov=src --cov-report=term --cov-report=html:reports/coverage_html -v

# 2. 查看HTML覆盖率报告
start reports/coverage_html/index.html  # Windows
```

### CI/CD集成命令

```powershell
# 并行执行 + 覆盖率 + JUnit XML报告
pytest tests/unit/ -n auto --cov=src --cov-report=term --cov-report=xml --cov-report=html --junitxml=reports/junit.xml -v
```

## 🔍 故障排查

### 如果测试失败

```powershell
# 1. 只运行失败的测试
pytest --lf -v  # last-failed

# 2. 详细输出
pytest tests/unit/xxx/test_xxx.py -v --tb=long

# 3. 进入调试模式
pytest tests/unit/xxx/test_xxx.py --pdb
```

### 如果覆盖率不足

```powershell
# 1. 查看详细覆盖率
pytest tests/unit/xxx/ --cov=src/xxx --cov-report=term-missing -v

# 2. 生成HTML报告，查看哪些行未覆盖
pytest tests/unit/xxx/ --cov=src/xxx --cov-report=html -v
```
