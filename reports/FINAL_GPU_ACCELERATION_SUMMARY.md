## 使用指南

### 基本使用
```python
from src.features.processors.gpu.gpu_technical_processor import GPUTechnicalProcessor

# 配置GPU处理器
config = {
    'use_gpu': True,
    'optimization_level': 'balanced',
    'gpu_threshold': 1000,
    'memory_limit': 0.8
}

processor = GPUTechnicalProcessor(config=config)

# 计算技术指标
data = pd.DataFrame(...)  # OHLC数据
result = processor.calculate_multiple_indicators_gpu(
    data, 
    ['sma', 'ema', 'rsi', 'macd', 'bollinger', 'atr']
)
```

### 最佳实践
1. **数据大小**: 建议10,000条以上使用GPU
2. **算法选择**: RSI和ATR优先使用GPU，EMA和MACD考虑CPU
3. **内存配置**: 使用保守模式（memory_limit=0.7）
4. **批量处理**: 多指标计算使用批量处理接口
