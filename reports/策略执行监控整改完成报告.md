## 六、使用指南

### 6.1 快速开始

```python
# 1. 初始化监控器

monitor = get_execution_monitor()
monitor.register_strategy(
    strategy_id="strategy_001",
    strategy_name="双均线策略",
    user_id="trader1"
)

# 2. 启动监控

# 3. 更新状态
monitor.update_strategy_status(
    strategy_id="strategy_001",
    status=ExecutionStatus.RUNNING
)
```

```python
    rule_id="custom_latency",
    threshold=500.0,
    severity=AlertSeverity.HIGH,
    metric_type="latency"

# 注册回调

)
```
