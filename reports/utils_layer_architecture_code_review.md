**文件信息**:
- 路径: src/utils/logger.py
- 行数: 20行
- 类型: 别名模块
- 功能: 提供向后兼容的导入路径

**文件内容**:
```python
"""
工具层日志模块（别名模块）
提供向后兼容的导入路径
"""

try:
    from src.infrastructure.utils.logging import get_logger, get_unified_logger
except ImportError:
    import logging
    
    def get_logger(name: str = None):
        """获取日志器"""
        return logging.getLogger(name or __name__)
    
    def get_unified_logger(name: str = None):
        """获取统一日志器"""
        return get_logger(name)

__all__ = ['get_logger', 'get_unified_logger']
```

**性质判断**: ✅ **别名模块**（非实现代码）

**问题分析**:
- 导入路径指向: `src.infrastructure.utils.logging`
- 本地实现: `src/utils/logging/logger.py`
- 导入路径不一致（可能是历史遗留）

**建议**:
- **选项1**: 删除根目录logger.py，更新导入路径
- **选项2**: 保留作为兼容层
- **推荐**: 选项1（清理根目录）
