```

AttributeError: 'RiskManager' object has no attribute 'assess_portfolio_risk'
AttributeError: 'RiskManager' object has no attribute 'calculate_var'
```

- **原因**: 测试期望的方法在实际类中不存在

#### **3. 断言逻辑问题**
```

AssertionError: assert 0 == 30
```
- **影响**: 实时监控和并发测试失败
- **原因**: 测试逻辑与实际实现不符

#### **4. 枚举类型不匹配**
```

```
- **影响**: 置信水平枚举比较失败
- **原因**: 枚举值比较方式不正确
