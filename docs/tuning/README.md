# 调优模块文档

## 📋 模块概述

调优模块 (`src/tuning/`) 负责模型调优、参数优化、自动化调参等功能。

## 🏗️ 模块结构

```
src/tuning/
├── __init__.py                    # 模块初始化
├── hyperparameter_tuner.py         # 超参数调优器
├── parameter_optimizer.py          # 参数优化器
├── auto_tuner.py                   # 自动调参器
└── tuning_utils.py                 # 调优工具
```
