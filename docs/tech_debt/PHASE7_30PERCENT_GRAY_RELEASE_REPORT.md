## 🔄 A/B测试机制

### 用户分群策略
```python
# 用户ID哈希分群算法
def get_user_group(user_id: int, total_groups: int = 100) -> int:
    """基于用户ID哈希确定分组"""
    return hash(str(user_id)) % total_groups

# 30%用户路由规则
def route_user_to_version(user_id: int) -> str:
    """路由用户到对应版本"""
    group = get_user_group(user_id)
    if group < 30:  # 0-29组 (30%)
        return "app_30percent"  # 新版本
    else:  # 30-99组 (70%)
        return "app_10percent"  # 旧版本
```
