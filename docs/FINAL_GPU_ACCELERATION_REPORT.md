## 使用指南

### 基本使用
```python
from src.features.processors.gpu.gpu_technical_processor import GPUTechnicalProcessor

# 创建GPU处理器
processor = GPUTechnicalProcessor()

# 计算单个指标
sma = processor.calculate_sma_gpu(data, window=20)
rsi = processor.calculate_rsi_gpu(data, window=14)

# 计算多个指标
indicators = ['sma', 'rsi', 'macd']
results = processor.calculate_multiple_indicators_gpu(data, indicators)
```

### 配置选项
```python
config = {
    'use_gpu': True,
    'batch_size': 1000,
    'memory_limit': 0.8,  # GPU内存使用限制
    'fallback_to_cpu': True
}
processor = GPUTechnicalProcessor(config)
```

### GPU信息监控
```python
gpu_info = processor.get_gpu_info()
print(f"GPU可用性: {gpu_info['available']}")
print(f"GPU设备: {gpu_info['device_name']}")
print(f"内存使用: {gpu_info['used_memory_gb']:.2f} GB")
```
