### 核心算法

**Redlock算法**:
1. 客户端生成唯一的锁ID
2. 尝试在所有Redis节点上获取锁
3. 计算获取锁成功的节点数量
4. 如果成功节点数 >= 多数派，则认为获取锁成功
5. 释放锁时，在所有节点上执行原子删除操作

**多数派计算**:
```python
quorum = len(redis_instances) // 2 + 1
```
