## 修复优先级

### 高优先级 (立即修复)

#### 1. 抽象类实现问题
**问题**: 多个抽象类缺少必要的方法实现
**影响**: 阻止测试运行

**需要修复的类**:
- `SignalGenerator` - 缺少 `generate_signals` 方法
- `CTPGateway` - 缺少多个抽象方法
- `XTPGateway` - 缺少多个抽象方法
- `OrderExecutor` - 缺少抽象方法
- `MarketImpactModel` - 构造函数参数问题

#### 2. 配置缺失问题
**问题**: 缺少必要的配置项
**影响**: 导致 KeyError

**需要修复的配置**:
- `settlement_engine.py` - 缺少 `commission` 配置
- `intelligent_rebalancing.py` - 缺少 `volatility` 因子权重

#### 3. 方法名不匹配
**问题**: 测试期望的方法名与实际实现不符
**影响**: AttributeError

**需要修复的方法**:
- `T1RestrictionChecker.check_sell_restriction`
- `PriceLimitChecker.check_price_limit`
- `STARMarketRuleChecker` 构造函数参数
- `CircuitBreaker.is_trading_allowed`

### 中优先级 (本周内修复)

#### 4. 异步支持问题
**问题**: 异步函数测试支持不足
**影响**: 异步测试无法运行

**解决方案**:
- 确保 `pytest-asyncio` 正确配置
- 修复异步测试装饰器

#### 5. 数值精度问题
**问题**: 浮点数比较精度问题
**影响**: 测试断言失败

**示例**:
```python
assert 0.19999999999998863 == 0.2  # 失败
```

### 低优先级 (下周修复)

#### 6. 编码问题

**影响**: 子进程测试失败

#### 7. 警告处理
**问题**: 各种FutureWarning和RuntimeWarning
**影响**: 测试输出混乱
