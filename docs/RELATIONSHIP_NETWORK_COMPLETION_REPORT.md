## 技术特性

### 核心算法
1. **相似度计算**:
   - 收益相似度（皮尔逊相关系数）
   - 风险相似度（夏普比率、最大回撤等）
   - 行为相似度（交易频率、持仓时间等）

2. **网络分析**:
   - 度中心性、介数中心性、接近中心性
   - 网络密度、聚类系数
   - 最短路径分析
   - 连通分量分析

3. **社区检测**:
   - Louvain算法
   - 模块化优化

4. **可视化**:
   - 网络图绘制
   - 节点大小映射到中心性
   - 边权重映射到颜色
   - 社区颜色编码
