# 集成模块文档

## 📋 模块概述

集成模块 (`src/ensemble/`) 负责模型集成、策略组合、集成优化等功能。

## 🏗️ 模块结构

```
src/ensemble/
├── __init__.py                    # 模块初始化
├── ensemble_manager.py             # 集成管理器
├── strategy_ensemble.py            # 策略集成
├── model_ensemble.py               # 模型集成
└── optimization.py                 # 集成优化
```
