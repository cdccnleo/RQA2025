# 适配器模块文档

## 📋 模块概述

适配器模块 (`src/adapters/`) 负责与第三方系统和服务的集成，提供统一的接口适配。

## 🏗️ 模块结构

```
src/adapters/
├── __init__.py                    # 模块初始化
└── miniqmt/                       # MiniQMT适配器
    ├── __init__.py                # MiniQMT模块初始化
    ├── adapter.py                 # MiniQMT适配器
    ├── connection.py              # 连接管理
    ├── data_handler.py            # 数据处理
    └── order_handler.py           # 订单处理
```
