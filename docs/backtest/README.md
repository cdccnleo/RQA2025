# 回测模块文档

## 📋 模块概述

回测模块 (`src/backtest/`) 提供策略回测功能，包括历史数据回测、性能分析、风险评估等。

## 🏗️ 模块结构

```
src/backtest/
├── __init__.py                    # 模块初始化
├── backtest_engine.py              # 回测引擎
├── performance_analyzer.py         # 性能分析器
├── risk_analyzer.py                # 风险分析器
└── report_generator.py             # 报告生成器
```
