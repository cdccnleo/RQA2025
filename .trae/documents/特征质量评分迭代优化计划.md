## 第一阶段: 数据质量因子 (P0)

### 1.1 数据质量评估维度

```python
# 数据质量因子计算
QUALITY_FACTORS = {
    'completeness': {  # 数据完整性
        'weight': 0.3,
        'threshold': 0.95,  # 缺失率 < 5%
        'calculation': lambda missing_ratio: max(0, 1 - missing_ratio / 0.05)
    },
    'stability': {  # 数据稳定性
        'weight': 0.25,
        'threshold': 0.95,  # 异常值比例 < 5%
        'calculation': lambda outlier_ratio: max(0, 1 - outlier_ratio / 0.05)
    },
    'calculation_success': {  # 计算成功率
        'weight': 0.25,
        'threshold': 0.99,  # 成功率 > 99%
        'calculation': lambda success_rate: success_rate
    },
    'time_coverage': {  # 时间跨度覆盖率
        'weight': 0.2,
        'threshold': 0.90,  # 覆盖率 > 90%
        'calculation': lambda coverage: coverage
    }
}
```

### 1.2 评分计算公式

```python
def calculate_final_quality_score(feature_name: str, data_quality_metrics: dict) -> float:
    """
    计算最终质量评分
    
    公式: final_score = base_score × quality_factor
    
    Args:
        feature_name: 特征名称
        data_quality_metrics: 数据质量指标
    
    Returns:
        最终质量评分
    """
    # 获取基础评分
    base_score = get_feature_quality_score(feature_name)
    
    # 计算数据质量因子
    quality_factor = calculate_quality_factor(data_quality_metrics)
    
    # 计算最终评分
    final_score = base_score * quality_factor
    
    # 确保评分在合理范围内
    return max(0.0, min(1.0, final_score))
```
