# 策略优化器批量删除功能计划

## 计划目标
为策略优化器(strategy-optimizer)页面的优化结果列表增加批量删除功能，提升用户体验。

## 功能需求

### 1. 前端界面
- 在优化结果表格每行添加复选框
- 添加全选/取消全选功能
- 添加批量删除按钮
- 显示选中数量
- 删除前确认对话框

### 2. 后端API
- 创建批量删除API端点
- 支持删除多个优化结果
- 同时删除PostgreSQL和文件系统中的数据
- 返回删除结果统计

### 3. 交互逻辑
- 选择/取消选择单个结果
- 全选/取消全选所有结果
- 批量删除确认
- 删除后刷新列表
- 显示操作成功/失败提示

## 实现步骤

### 第一阶段：后端API开发（30分钟）
1. 创建批量删除API端点 `/api/v1/strategy/optimization/results/batch-delete`
2. 实现批量删除逻辑
3. 同时删除PostgreSQL和文件系统数据
4. 添加错误处理和日志

### 第二阶段：前端界面开发（45分钟）
1. 在表格头部添加全选复选框
2. 在每行数据前添加选择复选框
3. 添加批量删除按钮（显示选中数量）
4. 实现选择/取消选择逻辑
5. 实现全选/取消全选逻辑

### 第三阶段：交互功能开发（30分钟）
1. 实现删除确认对话框
2. 调用批量删除API
3. 删除成功后刷新列表
4. 显示操作结果提示

### 第四阶段：测试验证（30分钟）
1. 测试单个删除功能
2. 测试批量删除功能
3. 测试全选删除功能
4. 验证数据一致性
