## 二、阶段二：数据安全修复 ✅

### 2.1 股票白名单验证
**文件**: `src/gateway/web/model_training_routes.py`

**修改内容**:
- 定义允许的股票代码白名单（100只主流股票）
- 在创建训练任务时验证股票代码
- 如果包含不支持的股票，返回400错误

**白名单示例**:
```python
ALLOWED_STOCKS: Set[str] = {
    '000001', '000002', '000063', '000100', '000333',  # 平安银行、万科A等
    '600000', '600009', '600016', '600028', '600030',  # 浦发银行、上海机场等
    # ... 共100只主流股票
}
```

**验证逻辑**:
```python
symbols = config.get("symbols", [])
if symbols:
    invalid_symbols = set(symbols) - ALLOWED_STOCKS
    if invalid_symbols:
        raise HTTPException(
            status_code=400,
            detail=f"包含不支持的股票代码: {list(invalid_symbols)}"
        )
```

### 2.2 日期范围限制
**文件**: `src/gateway/web/model_training_routes.py`

**修改内容**:
- 定义最大日期范围（5年）
- 验证开始日期和结束日期
- 如果超出范围，返回400错误
- 如果日期范围小于30天，给出警告

**代码示例**:
```python
MAX_DATE_RANGE_DAYS = 365 * 5  # 5年

time_range = config.get("time_range", {})
start_date_str = time_range.get("start_date")
end_date_str = time_range.get("end_date")

if start_date_str and end_date_str:
    start_date = datetime.strptime(start_date_str, "%Y-%m-%d")
    end_date = datetime.strptime(end_date_str, "%Y-%m-%d")
    days = (end_date - start_date).days
    
    if days > MAX_DATE_RANGE_DAYS:
        raise HTTPException(
            status_code=400,
            detail=f"日期范围不能超过{MAX_DATE_RANGE_DAYS}天（约5年）"
        )
    
    if days < 30:
        config["warning_messages"].append(
            f"日期范围仅{days}天，建议至少30天"
        )
```

### 2.3 改进效果
- ✅ 股票代码白名单验证
- ✅ 日期范围限制（最多5年）
- ✅ 数据安全性提升

**文件**: `src/gateway/web/training_job_executor.py`

**修改内容**:

- 实现归一化（MinMaxScaler）
- 实现标准化（StandardScaler）
- 记录预处理应用到结果

**代码示例**:
```python
preprocessing = config.get("preprocessing", {})
if preprocessing.get("normalize") or preprocessing.get("scale"):
    from sklearn.preprocessing import StandardScaler, MinMaxScaler
    
    numeric_cols = training_data.select_dtypes(include=['float64', 'int64']).columns
    numeric_cols = [col for col in numeric_cols if col not in ['symbol', 'date']]
    
    if preprocessing.get("normalize"):
        scaler = MinMaxScaler()
        training_data[numeric_cols] = scaler.fit_transform(training_data[numeric_cols])
        result["preprocessing_applied"].append("normalize")
    
    if preprocessing.get("scale"):
        scaler = StandardScaler()
        training_data[numeric_cols] = scaler.fit_transform(training_data[numeric_cols])
        result["preprocessing_applied"].append("scale")
```
