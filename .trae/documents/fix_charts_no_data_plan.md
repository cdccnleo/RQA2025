# 策略回测分析页面图表无数据展示问题修复计划

## 任务目标
修复策略回测分析页面（strategy-backtest.html）中，已有回测记录和交易记录的情况下，累计收益率走势和风险收益分布仪表盘无数据展示的问题。

## 问题现象
- 页面已有回测记录和交易记录
- 累计收益率走势仪表盘无数据展示
- 风险收益分布仪表盘无数据展示

## 可能原因分析

### 1. 数据传递问题
- 后端返回的回测结果数据格式不正确
- `equity_curve` 字段缺失或为空
- `returns` 或 `daily_returns` 字段缺失

### 2. 前端数据处理问题
- `updateCharts` 函数未被正确调用
- `updateReturnsChart` 函数处理数据逻辑有误
- `updateRiskReturnChart` 函数处理数据逻辑有误
- 图表初始化失败或重复初始化导致错误

### 3. 图表渲染问题
- Chart.js 画布已被占用（已修复）
- 数据格式不符合 Chart.js 要求
- 标签和数据长度不匹配

## 验证步骤

### 第一阶段：检查后端数据（10分钟）
1. 检查回测结果数据结构
2. 验证 `equity_curve` 字段是否存在且有数据
3. 验证 `total_return`、`annualized_return` 等字段

### 第二阶段：检查前端数据处理（15分钟）
1. 检查 `loadBacktestHistory` 函数是否正确调用 `updateCharts`
2. 检查 `updateReturnsChart` 函数的数据处理逻辑
3. 检查 `updateRiskReturnChart` 函数的数据处理逻辑
4. 添加调试日志，追踪数据流向

### 第三阶段：修复问题（15分钟）
1. 修复数据传递问题
2. 修复数据处理逻辑
3. 确保图表正确渲染

### 第四阶段：测试验证（10分钟）
1. 刷新页面，检查图表是否正常显示
2. 验证数据准确性
3. 检查浏览器控制台是否有错误

## 检查清单
- [ ] 后端返回的回测数据包含 `equity_curve` 字段
- [ ] `equity_curve` 字段包含有效的数值数组
- [ ] `updateCharts` 函数被正确调用
- [ ] `updateReturnsChart` 正确处理数据
- [ ] `updateRiskReturnChart` 正确处理数据
- [ ] 图表正常渲染显示
- [ ] 无 JavaScript 错误

## 预期结果
累计收益率走势和风险收益分布仪表盘能够正确显示回测数据。

## 风险与缓解
| 风险 | 缓解措施 |
|------|----------|
| 数据格式不一致 | 添加数据验证和转换逻辑 |
| 图表初始化失败 | 添加错误处理和重试机制 |
| 性能问题 | 优化数据处理和渲染逻辑 |
