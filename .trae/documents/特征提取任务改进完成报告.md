### 1.1 阶段一：数据安全修复 ✅

#### 股票白名单验证
**文件**: `src/gateway/web/feature_engineering_routes.py`

**修改内容**:
- 定义允许的股票代码白名单（100只主流股票）
- 在创建单股票特征任务时验证股票代码
- 在创建批量特征任务时验证所有股票代码
- 如果包含不支持的股票，返回400错误

**代码实现**:
```python
ALLOWED_STOCKS: Set[str] = {
    '000001', '000002', '000063', ...  # 100只主流股票
}

def _validate_symbol(symbol: str) -> None:
    if symbol and symbol not in ALLOWED_STOCKS:
        raise HTTPException(status_code=400, detail=f"不支持的股票代码: {symbol}")

def _validate_symbols(symbols: List[str]) -> None:
    if symbols:
        invalid_symbols = set(symbols) - ALLOWED_STOCKS
        if invalid_symbols:
            raise HTTPException(status_code=400, detail=f"包含不支持的股票代码: {list(invalid_symbols)}")
```

#### 日期范围限制
**文件**: `src/gateway/web/feature_engineering_routes.py`

**修改内容**:
- 定义最大日期范围（5年）
- 验证开始日期和结束日期
- 如果超出范围，返回400错误
- 如果日期范围小于30天，给出警告

**代码实现**:
```python
MAX_DATE_RANGE_DAYS = 365 * 5

def _validate_date_range(start_date: str, end_date: str) -> tuple:
    start = datetime.strptime(start_date, "%Y-%m-%d")
    end = datetime.strptime(end_date, "%Y-%m-%d")
    days = (end - start).days
    
    if days > MAX_DATE_RANGE_DAYS:
        raise HTTPException(status_code=400, detail=f"日期范围不能超过{MAX_DATE_RANGE_DAYS}天")
    
    warnings = []
    if days < 30:
        warnings.append(f"日期范围仅{days}天，建议至少30天")
    
    return warnings, days
```

```python
def _enforce_quality_checks(config: Dict[str, Any]) -> List[str]:
    warnings = []
    
    if not config.get("enable_quality_check"):
        logger.warning("特征质量检查未启用，根据量化交易系统要求强制启用")
        config["enable_quality_check"] = True
        warnings.append("特征质量检查已根据量化交易系统要求强制启用")
    
    if not config.get("enable_anomaly_detection"):
        logger.warning("异常检测未启用，根据量化交易系统要求强制启用")
        config["enable_anomaly_detection"] = True
        warnings.append("异常检测已根据量化交易系统要求强制启用")
    
    return warnings
```

**文件**: `src/gateway/web/feature_engineering_service.py`

**修改内容**:
- 定义最大特征数量（100个）
- 在特征计算后验证特征数量
- 如果超出限制，使用方差选择最重要的特征

**代码实现**:
```python
MAX_FEATURE_COUNT = 100
if len(feature_columns) > MAX_FEATURE_COUNT:
    logger.warning(f"特征数量({len(feature_columns)})超过限制({MAX_FEATURE_COUNT})，将选择最重要的特征")
    
    from sklearn.feature_selection import VarianceThreshold
    selector = VarianceThreshold(threshold=0.01)
    X_temp = combined_features[feature_columns]
    selector.fit(X_temp)
    
    variances = selector.variances_
    top_indices = variances.argsort()[-MAX_FEATURE_COUNT:][::-1]
    feature_columns = [feature_columns[i] for i in top_indices]
```
