# 数据采集监控页面问题检查计划

## 检查目标

检查 `data-collection-monitor.html` 页面存在的以下问题：
1. 运行中的任务仪表盘显示：加载失败: 加载失败
2. 采集记录趋势数据是否使用了模拟数据或硬编码
3. 性能瓶颈仪表盘source_1、source_2是否存在，与当前数据源配置不匹配
4. 数据源健康度仪表盘数据是否来自真实数据

## 问题分析

### 问题1：运行中的任务仪表盘显示"加载失败"

**位置**: `web-static/data-collection-monitor.html` 第 1885-1910 行

**代码分析**:
```javascript
async function loadRunningTasks() {
    try {
        const response = await fetch(`${API_BASE_URL}/api/v1/data/scheduler/dashboard`);
        if (!response.ok) throw new Error(`HTTP ${response.status}`);
        const result = await response.json();
        if (!result.success) {
            throw new Error(result.message || '加载失败');  // <-- 问题点
        }
        // ...
    } catch (error) {
        document.getElementById('running-tasks-table').innerHTML = `
            <tr>
                <td colspan="6" style="text-align: center; color: #dc3545; padding: 40px;">
                    <i class="fas fa-exclamation-circle"></i> 加载失败: ${error.message}
                </td>
            </tr>
        `;
    }
}
```

**可能原因**:
1. API返回的数据格式中缺少 `success` 字段
2. API返回的数据格式与前端期望不匹配
3. `generateRunningTasks` 函数处理数据时出错

**检查方法**:
1. 测试API `/api/v1/data/scheduler/dashboard` 返回的数据格式
2. 检查是否包含 `success` 字段
3. 检查 `unified_scheduler` 数据结构

**位置**: `src/gateway/web/datasource_routes.py` 第 3269-3318 行

**代码分析**:
```python

async def get_collection_trends(days: int = 7):

                "task_count": random.randint(5, 20)
            })

```

**问题确认**: ✅ **确实使用了模拟数据**
- 使用 `random.randint()` 生成随机数据
- 没有从数据库或实际采集记录中读取数据
- 注释明确说明"生成模拟趋势数据"

## 修复方案

### 修复1：运行中任务加载失败

**方案A**: 修改API返回格式，添加success字段
```python
@router.get("/api/v1/data/scheduler/dashboard")
async def get_scheduler_dashboard():
    # ...
    return {
        "success": True,  # 添加这一行
        "data": {
            "scheduler": scheduler_status,
            "performance": performance_metrics,
            "unified_scheduler": unified_stats
        },
        "timestamp": time.time()
    }
```

**方案B**: 修改前端，兼容无success字段的响应
```javascript
async function loadRunningTasks() {
    try {
        const response = await fetch(`${API_BASE_URL}/api/v1/data/scheduler/dashboard`);
        if (!response.ok) throw new Error(`HTTP ${response.status}`);
        const result = await response.json();
        
        // 兼容处理：如果无success字段或success为true，都继续处理
        if (result.success === false) {
            throw new Error(result.message || '加载失败');
        }
        
        // 直接使用result.data或result本身
        const data = result.data || result;
        const runningTasks = generateRunningTasks(data);
        displayRunningTasks(runningTasks);
    } catch (error) {
        // ...
    }
}
```
