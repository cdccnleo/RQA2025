# 监控页面性能优化与标签页重构计划

## 背景

当前量化交易系统监控页面（dashboard）作为监控首页，已集成多个仪表盘：
- 业务流程驱动监控
- 策略优化与执行
- 交易执行流程
- 风险控制流程
- 21层级架构状态监控
- 系统性能监控
- 数据流监控
- 数据管理层监控
- 数据源监控
- 告警和事件监控
- 系统健康总览

由于页面集成内容较多，导致加载缓慢，用户体验不佳。

## 目标

1. **优化仪表盘链接**：使用新窗口打开，避免切换仪表盘时需再次加载首页
2. **标签页重构**：使用标签页方式进行页面布局与组织，提升用户体验
3. **性能优化**：减少首页加载时间，实现按需加载

## 计划详情

### Phase 1: 仪表盘链接优化

#### 1.1 修改仪表盘卡片链接
- **文件**: `web-static/dashboard.html`

## 技术方案

### 标签页切换逻辑
```javascript
class DashboardTabs {
    constructor() {
        this.tabs = ['overview', 'strategy', 'trading', 'data', 'alert', 'architecture'];
        this.activeTab = 'overview';
        this.loadedTabs = new Set();
        this.cache = new Map();
    }
    
    async switchTab(tabId) {
        // 1. 保存当前标签页状态
        this.saveTabState();
        
        // 2. 切换标签页UI
        this.updateTabUI(tabId);
        
        // 3. 检查是否已加载
        if (this.loadedTabs.has(tabId)) {
            // 从缓存恢复
            this.restoreFromCache(tabId);
        } else {
            // 异步加载内容
            await this.loadTabContent(tabId);
            this.loadedTabs.add(tabId);
        }
        
        this.activeTab = tabId;
    }
    
    async loadTabContent(tabId) {
        const contentUrl = `/dashboard/content/${tabId}`;
        const response = await fetch(contentUrl);
        const html = await response.text();
        
        // 缓存内容
        this.cache.set(tabId, html);
        
        // 渲染到容器
        document.getElementById(`tab-${tabId}`).innerHTML = html;
        
        // 初始化该标签页的图表和组件
        this.initTabComponents(tabId);
    }
}
```
