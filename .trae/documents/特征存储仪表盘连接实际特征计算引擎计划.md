# 特征存储仪表盘连接实际特征计算引擎计划

## 目标

将特征存储仪表盘从显示硬编码模拟数据改为显示由 TechnicalProcessor 动态计算的实际特征数据。

## 当前状态分析

### 问题诊断

1. **硬编码数据**: `src/features/core/engine.py` 中的 `initialize_sample_data()` 方法提供静态模拟数据
2. **数据流断裂**: 特征计算引擎 (`TechnicalProcessor`) 与特征存储之间没有数据连接
3. **API返回模拟数据**: `/features/engineering/features` 端点返回硬编码特征

### 当前数据流

```
特征存储仪表盘
    ↓
GET /features/engineering/features
    ↓
get_features() [feature_engineering_service.py:481]
    ↓
get_feature_engine().get_features() [engine.py]
    ↓
initialize_sample_data() [engine.py:1071]
    ↓
硬编码的模拟数据
```

## 优化方案

### 方案一: 实时计算特征（推荐）

修改 `get_features()` 函数，在请求时实时计算特征。

**优点**:

* 数据始终最新

* 无需持久化存储

* 实现简单

**缺点**:

* 每次请求都计算，性能开销大

* 不适合大量特征

### 方案二: 任务驱动特征计算

当特征工程任务完成时，将计算结果保存到特征存储。

**优点**:

* 数据持久化

* 可追踪特征版本

* 支持历史查询

**缺点**:

* 实现复杂

* 需要修改任务执行流程

### 方案三: 混合方案（最终选择）

结合实时计算和缓存机制：

1. 首次请求时计算特征并缓存
2. 后续请求返回缓存数据
3. 定期刷新或手动触发重新计算

### 步骤 1: 修改 FeatureEngine 类

```python
# src/features/core/engine.py

class FeatureEngine:
    def __init__(self):
        # ... 现有代码 ...
        
        # 新增：初始化 TechnicalProcessor
        from src.features.processors.technical.technical_processor import TechnicalProcessor
        self.technical_processor = TechnicalProcessor()
        
        # 新增：特征缓存
        self._features_cache = {}
        self._cache_timestamp = None
        self._cache_ttl = 300  # 5分钟
    
    def get_features(self) -> List[Dict[str, Any]]:
        """获取特征列表 - 动态计算"""
        # 检查缓存是否有效
        if self._is_cache_valid():
            return self._features_cache.get('features', [])
        
        # 计算特征
        features = self._calculate_features()
        
        # 更新缓存
        self._features_cache['features'] = features
        self._cache_timestamp = time.time()
        
        return features
    
    def _calculate_features(self) -> List[Dict[str, Any]]:
        """使用 TechnicalProcessor 计算特征"""
        try:
            # 生成或获取股票数据
            data = self._get_sample_stock_data()
            
            # 定义要计算的指标
            indicators = ["sma", "rsi", "macd"]
            params = {
                "sma_periods": [20],
                "rsi_period": 14,
                "macd_fast": 12,
                "macd_slow": 26,
                "macd_signal": 9
            }
            
            # 计算指标
            results = self.technical_processor.calculate_multiple_indicators(
                data, indicators, params
            )
            
            # 转换为特征格式
            features = []
            for column in results.columns:
                feature = {
                    "name": column,
                    "display_name": self._get_display_name(column),
                    "type": "technical",
                    "importance": 0.7,
                    "correlation": 0.6,
                    "quality_score": 1.0,
                    "missing_values": results[column].isna().mean(),
                    "created_at": time.time(),
                    "updated_at": time.time()
                }
                features.append(feature)
            
            return features
        except Exception as e:
            logger.error(f"计算特征失败: {e}")
            return []
    
    def _get_sample_stock_data(self) -> pd.DataFrame:
        """生成示例股票数据"""
        import pandas as pd
        import numpy as np
        
        # 生成100天的示例数据
        dates = pd.date_range(end=pd.Timestamp.now(), periods=100, freq='D')
        np.random.seed(42)
        
        # 生成随机价格序列
        prices = 100 + np.cumsum(np.random.randn(100) * 2)
        
        data = pd.DataFrame({
            'open': prices * 0.99,
            'high': prices * 1.02,
            'low': prices * 0.98,
            'close': prices,
            'volume': np.random.randint(1000000, 10000000, 100)
        }, index=dates)
        
        return data
    
    def _get_display_name(self, column: str) -> str:
        """获取特征的显示名称"""
        display_names = {
            'sma_20': '20日均线',
            'rsi': '14日RSI',
            'macd': 'MACD',
            'macd_signal': 'MACD信号',
            'macd_histogram': 'MACD柱状图'
        }
        return display_names.get(column, column)
    
    def _is_cache_valid(self) -> bool:
        """检查缓存是否有效"""
        if not self._cache_timestamp:
            return False
        return (time.time() - self._cache_timestamp) < self._cache_ttl
    
    def refresh_features(self):
        """手动刷新特征"""
        self._cache_timestamp = None
        return self.get_features()
```

```python
# src/gateway/web/feature_engineering_routes.py

@router.post("/features/engineering/refresh")
async def refresh_features() -> Dict[str, Any]:
    """手动刷新特征计算"""
    try:
        engine = get_feature_engine()
        if engine and hasattr(engine, 'refresh_features'):
            features = engine.refresh_features()
            return {
                "success": True,
                "message": f"成功刷新 {len(features)} 个特征",
                "feature_count": len(features)
            }
        else:
            return {
                "success": False,
                "message": "特征引擎不支持刷新功能"
            }
    except Exception as e:
        logger.error(f"刷新特征失败: {e}")
        raise HTTPException(status_code=500, detail=f"刷新特征失败: {str(e)}")
```

### 特征属性

* `name`: 特征标识符（如 `sma_20`, `rsi`, `macd_signal`）

* `display_name`: 中文显示名称（如 `20日均线`, `14日RSI`, `MACD信号`）

* `type`: 特征类型（`technical`）

* `importance`: 重要性评分（基于计算结果）

* `correlation`: 相关性评分

* `quality_score`: 质量评分（基于缺失值比例）

* `missing_values`: 缺失值比例

* `created_at`: 创建时间

* `updated_at`: 更新时间
