# 自定义评分去重优化计划

## 目标
优化自定义特征质量评分功能，按特征类型去重显示，自定义评分应用于所选特征的所有股票代码。

## 背景
- 当前特征按不同股票代码分别存储（如 sma_688702, sma_002837）
- 用户希望相同特征类型（如 SMA）的所有股票使用相同的自定义评分
- 需要在UI中按特征类型去重显示，简化用户操作

## 当前问题
1. **重复显示**: 相同特征类型（如 SMA）会因不同股票代码而显示多次
2. **操作繁琐**: 用户需要为每个股票代码的相同特征分别设置评分
3. **不符合预期**: 用户期望设置一次 SMA 的评分，就应用到所有股票的 SMA

## 优化方案

### 方案1: 前端去重显示（推荐）

**思路**: 在前端渲染时对特征列表进行去重，按特征类型（feature_type）分组显示

**优点**:
- 无需修改后端API
- 实现简单快速
- 兼容现有数据结构

**实现步骤**:
1. 提取唯一的特征类型列表
2. 按特征类型分组显示
3. 显示该特征类型的所有股票数量
4. 自定义评分应用到该特征类型的所有股票

### 方案2: 后端API优化

**思路**: 后端提供按特征类型聚合的API，返回唯一的特征类型列表和统计信息

**优点**:
- 减少前端处理逻辑
- 数据结构更清晰
- 便于后续扩展

**缺点**:
- 需要修改后端API
- 工作量较大

**实现步骤**:
1. 创建新的API端点 `/features/engineering/quality/config/by-type`
2. 按 feature_type 分组聚合数据
3. 返回特征类型、股票数量、默认评分等信息
4. 前端调用新API显示

## 技术实现细节

### 特征类型提取逻辑
```javascript
function getFeatureType(feature) {
    // 优先使用 feature_type 字段
    if (feature.feature_type) {
        return feature.feature_type;
    }
    // 从名称中提取（如 sma_688702 -> SMA）
    return feature.name.split('_')[0].toUpperCase();
}
```

### 按特征类型分组显示
```javascript
function groupFeaturesByType(features) {
    const groups = {};
    features.forEach(feature => {
        const type = getFeatureType(feature);
        if (!groups[type]) {
            groups[type] = {
                type: type,
                features: [],
                symbols: new Set(),
                defaultScore: feature.quality_score
            };
        }
        groups[type].features.push(feature);
        groups[type].symbols.add(feature.symbol);
    });
    return groups;
}
```

### 自定义评分应用逻辑
```javascript
// 保存自定义评分时，应用到该类型的所有特征
async function saveQualityConfigByType(featureType, customScore, reason) {
    // 获取该类型的所有特征
    const featuresOfType = allFeatures.filter(f => getFeatureType(f) === featureType);
    
    // 批量保存配置
    const configs = featuresOfType.map(f => ({
        feature_name: f.name,
        custom_score: customScore,
        reason: reason
    }));
    
    await fetch('/api/v1/features/engineering/quality/config/batch', {
        method: 'POST',
        headers: { 'Content-Type': 'application/json' },
        body: JSON.stringify({ configs })
    });
}
```
