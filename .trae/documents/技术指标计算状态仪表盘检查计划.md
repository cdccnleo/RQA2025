# 技术指标计算状态仪表盘检查计划

## 问题描述
特征工程监控页面 (feature-engineering-monitor.html) 的技术指标计算状态仪表盘仅显示 MA 计算次数，其他指标（如 EMA、RSI、MACD 等）没有显示。

## 检查目标
1. 检查技术指标计算状态的数据来源
2. 检查仪表盘的数据绑定逻辑
3. 检查后端 API 返回的数据结构
4. 检查前端展示逻辑

## 检查内容

### 1. 前端页面检查
- **文件**: `web-static/feature-engineering-monitor.html`
- **检查项**:
  - 技术指标计算状态仪表盘的 HTML 结构
  - 数据绑定逻辑
  - JavaScript 数据处理代码

### 2. 后端 API 检查
- **文件**: `src/gateway/web/feature_engineering_routes.py`
- **检查项**:
  - 技术指标计算状态 API 端点
  - 返回的数据结构
  - 数据统计逻辑

### 3. 数据存储检查
- **文件**: `src/gateway/web/feature_task_persistence.py`
- **检查项**:
  - 任务指标统计存储
  - 指标计算次数记录

### 4. 任务执行检查
- **文件**: `src/gateway/web/feature_task_executor.py`
- **检查项**:
  - 指标计算逻辑
  - 统计更新逻辑

## 实施步骤

### 步骤1: 检查前端仪表盘代码
1. 查看 feature-engineering-monitor.html 中的技术指标计算状态仪表盘
2. 检查数据绑定和展示逻辑
3. 检查是否有数据过滤或限制

### 步骤2: 检查后端 API
1. 查看技术指标计算状态 API 端点
2. 检查返回的数据结构
3. 检查数据统计逻辑

### 步骤3: 检查数据存储
1. 查看数据库中指标计算统计
2. 检查是否有其他指标的数据

### 步骤4: 检查任务执行
1. 查看特征任务执行器的指标计算逻辑
2. 检查统计更新是否正确

## 文件清单
- `web-static/feature-engineering-monitor.html`
- `src/gateway/web/feature_engineering_routes.py`
- `src/gateway/web/feature_task_persistence.py`
- `src/gateway/web/feature_task_executor.py`
