# 修复数据一致性计划

## 目标
修复 `indicator_calculation_tracker.py` 的加载逻辑，确保内存数据与持久化文件 `indicator_calculations.json` 的数据一致性。

## 问题诊断

### 当前问题
1. **内存数据与持久化文件不一致**: 内存显示 0 次计算，但文件中有 26 次计算记录
2. **容器重启后数据丢失**: 跟踪器重新初始化时，`_calculations` 字典未正确加载文件数据
3. **单例模式问题**: `get_indicator_calculation_tracker()` 返回的实例可能未正确加载历史数据

### 根本原因
- `_load_calculations()` 方法可能在初始化时未正确执行
- 文件路径或权限问题导致读取失败
- JSON 解析错误或数据格式不匹配
- 多线程/多进程环境下的并发问题

## 代码修改详情

### 修改 1: 增强 `_load_calculations()`
```python
def _load_calculations(self) -> None:
    """从文件加载计算记录 - 增强版"""
    try:
        if os.path.exists(self.persistence_file):
            with open(self.persistence_file, 'r', encoding='utf-8') as f:
                data = json.load(f)
                
            # 验证数据格式
            for indicator_name, calc_data in data.items():
                if isinstance(calc_data, dict) and 'count' in calc_data:
                    self._calculations[indicator_name] = {
                        "count": calc_data.get("count", 0),
                        "last_computed": calc_data.get("last_computed"),
                        "task_history": calc_data.get("task_history", [])
                    }
                    
            logger.info(f"从 {self.persistence_file} 加载了 {len(data)} 个指标计算记录")
        else:
            logger.warning(f"持久化文件不存在: {self.persistence_file}")
    except Exception as e:
        logger.error(f"加载计算记录失败: {e}", exc_info=True)
```

### 修改 2: 增强 `_save_calculations()`
```python
def _save_calculations(self) -> None:
    """保存计算记录到文件 - 增强版"""
    try:
        # 确保目录存在
        os.makedirs(os.path.dirname(self.persistence_file), exist_ok=True)
        
        # 原子性写入
        temp_file = f"{self.persistence_file}.tmp"
        with open(temp_file, 'w', encoding='utf-8') as f:
            json.dump(dict(self._calculations), f, ensure_ascii=False, indent=2)
        
        # 重命名（原子操作）
        os.replace(temp_file, self.persistence_file)
        
        logger.debug(f"计算记录已保存到 {self.persistence_file}")
    except Exception as e:
        logger.error(f"保存计算记录失败: {e}", exc_info=True)
```

### 修改 3: 修复单例模式
```python
# 全局实例
_calculation_tracker_instance = None
_instance_lock = threading.Lock()

def get_indicator_calculation_tracker() -> IndicatorCalculationTracker:
    """获取指标计算跟踪器实例（单例模式）"""
    global _calculation_tracker_instance
    
    if _calculation_tracker_instance is None:
        with _instance_lock:
            if _calculation_tracker_instance is None:
                _calculation_tracker_instance = IndicatorCalculationTracker()
                logger.info("创建新的指标计算跟踪器实例")
    
    return _calculation_tracker_instance
```
