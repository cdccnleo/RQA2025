# 自定义评分实施计划

## 目标
实施自定义评分功能，允许用户为特定特征设置评分，并在特征工程监控页面增加交互式界面。

## 背景
- 用户可能需要根据实际使用经验调整特征评分
- 需要在前端提供直观的交互界面
- 自定义评分应该优先于系统默认评分

## 实施方案

### 阶段1: 后端API开发

#### 1.1 数据库表设计
**表名**: `user_feature_quality_config`

```sql
CREATE TABLE user_feature_quality_config (
    config_id SERIAL PRIMARY KEY,
    user_id VARCHAR(255) NOT NULL,
    feature_name VARCHAR(255) NOT NULL,
    custom_score FLOAT NOT NULL CHECK (custom_score >= 0 AND custom_score <= 1),
    reason TEXT,
    is_active BOOLEAN DEFAULT TRUE,
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    updated_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    UNIQUE(user_id, feature_name)
);

CREATE INDEX idx_user_feature_config_user_id ON user_feature_quality_config(user_id);
CREATE INDEX idx_user_feature_config_feature ON user_feature_quality_config(feature_name);
CREATE INDEX idx_user_feature_config_user_feature ON user_feature_quality_config(user_id, feature_name);
```

#### 1.2 API端点设计

**1. 获取用户自定义评分配置**
- 端点: `GET /api/v1/features/quality/config`
- 参数: `user_id` (可选，默认当前用户)
- 返回: 自定义评分配置列表

**2. 设置自定义评分**
- 端点: `POST /api/v1/features/quality/config`
- 参数: `feature_name`, `custom_score`, `reason`
- 返回: 创建的配置信息

**3. 更新自定义评分**
- 端点: `PUT /api/v1/features/quality/config/{config_id}`
- 参数: `custom_score`, `reason`
- 返回: 更新后的配置信息

**4. 删除自定义评分**
- 端点: `DELETE /api/v1/features/quality/config/{config_id}`
- 返回: 删除成功信息

**5. 重置为默认评分**
- 端点: `POST /api/v1/features/quality/config/{config_id}/reset`
- 返回: 重置成功信息

**6. 批量设置自定义评分**
- 端点: `POST /api/v1/features/quality/config/batch`
- 参数: `configs` (数组)
- 返回: 批量操作结果

## 技术实现细节

### 评分优先级逻辑

```python
def get_feature_quality_score_with_custom(
    feature_name: str,
    user_id: str = None
) -> float:
    """
    获取特征质量评分（支持用户自定义）
    
    优先级:
    1. 用户自定义评分（如果存在且有效）
    2. 系统默认评分
    """
    # 1. 检查用户自定义评分
    if user_id:
        custom_config = get_user_custom_config(user_id, feature_name)
        if custom_config and custom_config.is_active:
            return custom_config.custom_score
    
    # 2. 返回系统默认评分
    return get_feature_quality_score(feature_name)
```

### 前端状态管理

```javascript
// 自定义评分状态
const customQualityConfig = {
    // 当前用户的自定义配置
    configs: {},
    
    // 加载配置
    async loadConfigs() {
        const response = await fetch('/api/v1/features/quality/config');
        const data = await response.json();
        this.configs = data.configs.reduce((acc, config) => {
            acc[config.feature_name] = config;
            return acc;
        }, {});
    },
    
    // 获取评分（优先自定义）
    getScore(featureName) {
        if (this.configs[featureName]) {
            return this.configs[featureName].custom_score;
        }
        return getDefaultScore(featureName);
    },
    
    // 设置自定义评分
    async setCustomScore(featureName, score, reason) {
        const response = await fetch('/api/v1/features/quality/config', {
            method: 'POST',
            headers: { 'Content-Type': 'application/json' },
            body: JSON.stringify({ feature_name: featureName, custom_score: score, reason })
        });
        
        if (response.ok) {
            const config = await response.json();
            this.configs[featureName] = config;
            return true;
        }
        return false;
    }
};
```
