# 量化交易系统模型训练改进评估计划

## 评估目标

全面分析评估量化交易系统中模型训练的以下三个关键事项：
1. 模型训练回退机制（真实训练失败时不应进行模拟训练）
2. PyTorch支持（LSTM/CNN/Transformer需要PyTorch）
3. 训练数据来源（应使用已采集的历史真实数据，而非默认股票代码000001）

## 1. 模型训练回退机制评估

### 当前实现

**文件**: `src/gateway/web/training_job_executor.py` 第548-569行

```python
try:
    # 尝试使用真实模型训练器
    from src.ml.real_model_trainer import get_real_model_trainer
    real_trainer = get_real_model_trainer()
    # ... 真实训练逻辑 ...
    
except Exception as e:
    logger.warning(f"真实训练失败，回退到模拟训练: {e}")
    
    # 回退到模拟训练
    epochs = config.get("epochs", 100)
    for epoch in range(1, epochs + 1):
        await asyncio.sleep(0.05)
        progress = int((epoch / epochs) * 100)
        # ... 模拟训练逻辑 ...
    
    result["accuracy"] = 0.85
    result["loss"] = 0.20
```

### 问题分析

**问题**: 真实训练失败时自动回退到模拟训练

**风险**:
1. **误导用户**: 用户以为进行的是真实训练，实际得到的是模拟结果
2. **模型不可用**: 模拟训练的模型没有实际预测能力
3. **数据污染**: 模拟结果可能污染模型评估指标
4. **信任危机**: 用户无法区分真实模型和模拟模型

### 建议方案

**方案1: 失败即停止（推荐）**
```python
try:
    # 真实训练
    training_result = real_trainer.train_model(...)
    if training_result.get('status') != 'completed':
        raise Exception(f"训练失败: {training_result.get('error')}")
    
except Exception as e:
    logger.error(f"真实训练失败: {e}")
    # 更新任务状态为失败
    await self._update_job_status(
        original_job_id, "failed",
        error_message=f"训练失败: {str(e)}"
    )
    # 不返回任何结果，让调用者处理失败
    raise
```

**方案2: 用户确认回退**
```python
except Exception as e:
    logger.warning(f"真实训练失败: {e}")
    
    # 检查是否允许模拟训练（通过配置或用户确认）
    if config.get('allow_fallback_to_mock', False):
        logger.info("用户允许回退到模拟训练")
        # 执行模拟训练，但明确标记为模拟
        result["is_mock"] = True
        result["mock_reason"] = str(e)
    else:
        # 失败停止
        raise Exception(f"真实训练失败，未允许模拟训练: {e}")
```

**方案3: 标记模拟结果**
```python
except Exception as e:
    logger.warning(f"真实训练失败，使用模拟训练: {e}")
    
    # 执行模拟训练
    # ... 模拟训练逻辑 ...
    
    # 明确标记为模拟结果
    result["is_mock"] = True
    result["mock_reason"] = str(e)
    result["accuracy"] = 0.85
    result["warning"] = "此结果为模拟训练，不可用于实际交易"
```

### 推荐方案

**推荐方案1: 失败即停止**

理由：
- 量化交易对模型准确性要求极高，模拟模型不可接受
- 失败即停止可以及时发现问题并修复
- 避免用户误用模拟模型进行实际交易
- 符合量化交易系统的高可靠性要求

实现LSTM模型示例:
```python
import torch
import torch.nn as nn
from torch.utils.data import DataLoader, TensorDataset

class LSTMModel(nn.Module):
    def __init__(self, input_size, hidden_size, num_layers, output_size):
        super(LSTMModel, self).__init__()
        self.hidden_size = hidden_size
        self.num_layers = num_layers
        self.lstm = nn.LSTM(input_size, hidden_size, num_layers, batch_first=True)
        self.fc = nn.Linear(hidden_size, output_size)
        self.sigmoid = nn.Sigmoid()
    
    def forward(self, x):
        h0 = torch.zeros(self.num_layers, x.size(0), self.hidden_size).to(x.device)
        c0 = torch.zeros(self.num_layers, x.size(0), self.hidden_size).to(x.device)
        out, _ = self.lstm(x, (h0, c0))
        out = self.fc(out[:, -1, :])
        out = self.sigmoid(out)
        return out

def train_lstm(X_train, y_train, X_test, y_test, config):
    # 数据准备
    X_train_tensor = torch.FloatTensor(X_train.values).unsqueeze(1)
    y_train_tensor = torch.FloatTensor(y_train.values)
    
    # 模型初始化
    model = LSTMModel(
        input_size=X_train.shape[1],
        hidden_size=config.get('hidden_size', 64),
        num_layers=config.get('num_layers', 2),
        output_size=1
    )
    
    # 训练
    criterion = nn.BCELoss()
    optimizer = torch.optim.Adam(model.parameters(), lr=config.get('learning_rate', 0.001))
    
    for epoch in range(config.get('epochs', 100)):
        # 前向传播
        outputs = model(X_train_tensor)
        loss = criterion(outputs.squeeze(), y_train_tensor)
        
        # 反向传播
        optimizer.zero_grad()
        loss.backward()
        optimizer.step()
        
        if epoch % 10 == 0:
            logger.info(f'Epoch [{epoch}/{config.get("epochs", 100)}], Loss: {loss.item():.4f}')
    
    return model
```

async def _load_training_data(self, config: Dict[str, Any]) -> Optional[pd.DataFrame]:
    # 查询数据库中所有已采集的股票
    conn = get_db_connection()
    cursor = conn.cursor()
    cursor.execute("""
        SELECT DISTINCT symbol FROM akshare_stock_data
        WHERE date >= %s AND date <= %s
    """, (config.get('start_date'), config.get('end_date')))
    symbols = [row[0] for row in cursor.fetchall()]
    cursor.close()
    return_db_connection(conn)
    
    # 加载所有股票数据
    all_data = []
    for symbol in symbols:
        df = await self._load_stock_data(symbol, config)
        if df is not None and len(df) >= 100:
            df['symbol'] = symbol
            all_data.append(df)
    
    if len(all_data) < 5:  # 至少需要5只股票
        logger.warning(f"可用股票数量不足: {len(all_data)}")
        return None
    
    return pd.concat(all_data, ignore_index=True)
