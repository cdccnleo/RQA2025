# 修复策略优化器页面优化结果数据及增加删除功能计划

## 任务目标
检查策略优化器(strategy-optimizer)页面中的优化结果，修复参数组合、得分、夏普比率、总收益、最大回撤等数据展示问题，并增加删除操作功能。

## 当前问题分析

### 问题1：优化结果数据展示
- 参数组合、得分、夏普比率、总收益、最大回撤等数据可能显示不正确或为空
- 需要检查前端展示逻辑和后端数据返回格式

### 问题2：缺少删除操作
- 优化结果列表中没有删除按钮或删除功能
- 需要增加删除单条优化结果的功能
- 需要增加批量删除功能（可选）

## 检查步骤

### 第一阶段：检查前端页面（10分钟）
1. 检查 strategy-optimizer.html 页面中的优化结果展示区域
2. 查看优化结果数据的渲染逻辑
3. 检查数据字段映射是否正确
4. 查看是否有删除按钮或相关操作

### 第二阶段：检查后端API（10分钟）
1. 检查获取优化结果的API端点
2. 验证返回的数据格式和字段
3. 检查是否有删除优化结果的API
4. 如果没有，需要新增删除API

### 第三阶段：修复数据展示（15分钟）
1. 修复前端数据展示逻辑
2. 确保所有字段正确显示
3. 添加数据格式化（如百分比、小数位等）

### 第四阶段：实现删除功能（15分钟）
1. 在后端增加删除优化结果的API
2. 在前端增加删除按钮
3. 实现删除确认对话框
4. 添加删除后的页面刷新

## 预期结果
- 优化结果数据正确显示（参数组合、得分、夏普比率、总收益、最大回撤）
- 每条优化结果都有删除按钮
