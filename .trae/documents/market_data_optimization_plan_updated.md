**关键组件**:
```

  ├── _convert_data() - 数据格式转换
  └── save_to_database() - 写入 PostgreSQL

数据采集调度器: DataCollectionScheduler
  ├── start() - 启动调度器
  ├── _scheduler_loop() - 主调度循环（每60秒检查）
  ├── _check_market_regime() - 市场状态检查
  ├── _prioritize_sources_intelligent() - 智能优先级排序
  └── _start_collection_task() - 启动采集任务
```

**启动流程**:
```
API Lifespan (api.py:754)
    ↓
start_data_collection_scheduler(startup_path="lifespan_direct")
    ↓
DataCollectionScheduler.start()
    ↓
_scheduler_loop() [每60秒检查一次]
    ↓
智能调度 → 数据采集 → PostgreSQL
```
