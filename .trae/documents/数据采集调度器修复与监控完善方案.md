## 🎯 问题背景

### 当前问题

1. **数据采集完成后未自动更新最后测试时间**
   - 数据源配置中的 `last_test` 字段仅在手动测试时更新
   - 数据采集成功完成后未同步更新该字段
   - 导致无法准确判断数据源的实际使用状态

2. **缺乏数据采集状态监控**
   - 无法深入检查当前数据采集配置是否已进行数据采集
   - 缺少数据采集成功/失败的统计信息
   - 无法追踪数据采集的历史记录

3. **特征存储仪表盘显示问题**（关联问题）
   - 特征工程监控页面的特征存储仪表盘仅显示硬编码的特征名称
   - 未显示实际的特征名称（如 `SMA_5`、`EMA_10` 等）

    # 新增字段 ⭐

#### 1.2 修改数据采集服务

在 `data_collection_service.py` 中添加采集完成回调：

```python
async def on_collection_complete(
    source_id: str,
    success: bool,
    record_count: int,
    error_message: str = None
):
    """
    数据采集完成回调函数
    
    Args:
        source_id: 数据源ID
        success: 是否成功
        record_count: 采集记录数
        error_message: 错误信息
    """
    from src.gateway.web.data_source_config_manager import get_data_source_config_manager
    
    config_manager = get_data_source_config_manager()
    
    updates = {
        "last_collection": datetime.now().strftime("%Y-%m-%d %H:%M:%S"),
        "collection_status": "success" if success else "failed",
        "collection_count": record_count if success else 0,
        "collection_errors": 0 if success else 1
    }
    
    if error_message:
        updates["last_error"] = error_message[:200]
    
    config_manager.update_data_source(source_id, updates)
```

#### 1.3 修改统一调度器

在 `unified_scheduler.py` 中添加任务完成事件处理：

```python
def _on_task_complete(self, task_id: str, result: Dict[str, Any]):
    """
    任务完成处理
    
    Args:
        task_id: 任务ID
        result: 任务结果
    """
    task_type = result.get("task_type")
    
    if task_type == TaskType.DATA_COLLECTION:
        source_id = result.get("source_id")
        success = result.get("success", False)
        record_count = result.get("record_count", 0)
        error_message = result.get("error_message")
        
        # 异步更新数据源状态
        asyncio.create_task(
            on_collection_complete(
                source_id=source_id,
                success=success,
                record_count=record_count,
                error_message=error_message
            )
        )
```

### 方案2: 数据采集监控仪表盘增强

#### 2.1 新增监控API端点

```python
@router.get("/api/v1/data/sources/collection-status")
async def get_data_sources_collection_status():
    """
    获取所有数据源的采集状态
    
    返回:
        - 每个数据源的最后采集时间
        - 采集状态（成功/失败/未采集）
        - 采集统计信息
    """
    from src.gateway.web.data_source_config_manager import get_data_source_config_manager
    
    config_manager = get_data_source_config_manager()
    sources = config_manager.get_data_sources()
    
    status_list = []
    for source in sources:
        status_list.append({
            "id": source.get("id"),
            "name": source.get("name"),
            "type": source.get("type"),
            "enabled": source.get("enabled", True),
            "last_test": source.get("last_test"),
            "last_collection": source.get("last_collection"),
            "collection_status": source.get("collection_status", "未采集"),
            "collection_count": source.get("collection_count", 0),
            "total_collections": source.get("total_collections", 0),
            "total_records": source.get("total_records", 0),
            "status": _calculate_status(source)
        })
    
    return {
        "success": True,
        "data": status_list,
        "summary": {
            "total": len(status_list),
            "collected": len([s for s in status_list if s.get("last_collection")]),
            "not_collected": len([s for s in status_list if not s.get("last_collection")]),
            "success": len([s for s in status_list if s.get("collection_status") == "success"]),
            "failed": len([s for s in status_list if s.get("collection_status") == "failed"])
        }
    }


def _calculate_status(source: Dict) -> str:
    """
    计算数据源综合状态
    
    规则:
    1. 如果未启用 -> "已禁用"
    2. 如果从未采集 -> "待采集"
    3. 如果最后采集失败 -> "采集失败"
    4. 如果最后采集成功但超过24小时 -> "待更新"
    5. 如果最后采集成功且在24小时内 -> "正常"
    """
    if not source.get("enabled", True):
        return "已禁用"
    
    last_collection = source.get("last_collection")
    if not last_collection:
        return "待采集"
    
    collection_status = source.get("collection_status")
    if collection_status == "failed":
        return "采集失败"
    
    # 检查采集时间
    try:
        last_time = datetime.strptime(last_collection, "%Y-%m-%d %H:%M:%S")
        hours_ago = (datetime.now() - last_time).total_seconds() / 3600
        
        if hours_ago > 24:
            return "待更新"
        return "正常"
    except:
        return "未知"
```

#### 3.2 修改特征任务执行器

在 `feature_task_executor.py` 中添加特征保存逻辑：

```python
async def _save_features_to_store(
    task_id: str,
    features: List[Dict[str, Any]]
):
    """
    保存特征到特征存储表
    
    Args:
        task_id: 任务ID
        features: 特征列表
    """
    from src.gateway.web.postgresql_persistence import get_db_connection, return_db_connection
    
    conn = get_db_connection()
    if not conn:
        return False
    
    try:
        cursor = conn.cursor()
        
        for feature in features:
            feature_id = f"{task_id}_{feature['name']}"
            
            cursor.execute("""
                INSERT INTO feature_store 
                (feature_id, task_id, feature_name, feature_type, parameters, symbol)
                VALUES (%s, %s, %s, %s, %s, %s)
                ON CONFLICT (feature_id) DO UPDATE SET
                    feature_name = EXCLUDED.feature_name,
                    feature_type = EXCLUDED.feature_type,
                    parameters = EXCLUDED.parameters
            """, (
                feature_id,
                task_id,
                feature.get("name"),
                feature.get("type"),
                json.dumps(feature.get("parameters", {})),
                feature.get("symbol")
            ))
        
        conn.commit()
        return True
        
    finally:
        return_db_connection(conn)
```
